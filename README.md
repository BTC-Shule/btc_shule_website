# 🌍 BTC Shule Website

This is the official website for **BTC Shule**, a Bitcoin-powered educational and community initiative based in Burundi.  
The site is built with **Next.js**, designed to share BTC Shule’s mission, resources, and updates with the global Bitcoin community.

---

## 🚀 About BTC Shule

BTC Shule is building a **Bitcoin-powered circular economy** in Winteko, Burundi.  
Our mission is to empower communities through **financial literacy, renewable energy, and economic autonomy** using Bitcoin.  

The project includes:
- 🏫 **Education Hub** – Digital & physical classes in Kirundi and French.  
- 🛍️ **Merchant Onboarding** – Helping local businesses accept Bitcoin.  
- 🤝 **Community Building** – Creating a self-sustaining Bitcoin economy.  

---

## 🛠️ Tech Stack

- [Next.js](https://nextjs.org/) – React framework for server-side rendering and static site generation.
- [TypeScript](https://www.typescriptlang.org/) – Type-safe JavaScript.
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework.
- [Framer Motion](https://www.framer.com/motion/) – Animations.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or newer)
- npm, yarn, pnpm, or bun

### Installation

### 1. Clone the Repository
```bash
git clone https://github.com/<your-org-or-username>/btc-shule-website.git
cd btc-shule-website
```

### 2. Install Dependencies
```bash
npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
```

### 3. Run Development Server
```bash
npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
```

Visit http://localhost:3000 to see the site locally.

### 4. Build for Production
```bash
npm run build
npm run start
```
### 📂 Project Structure
```bash
btc-shule-website/
├── public/        # Static assets (images, icons, etc.)
├── src/
│   ├── app/       # Next.js app directory
│   ├── components/ # Reusable UI components
│   └── utils/     # Helpers & utilities
└── README.md
```
